
# 🚖 Uber Ride Analysis Dashboard | Power BI Project

## 📌 Project Overview

This project presents an **interactive Uber Ride Analysis Dashboard** developed using **Microsoft Power BI**.  
The dashboard provides insights into **ride bookings, revenue trends, cancellations, distance traveled, and customer & driver ratings**.

The aim of this project is to analyze ride data and transform raw data into **meaningful business insights through data visualization**.

Uber is a global mobility platform connecting riders and drivers through a technology-driven marketplace operating in more than **70 countries** and generating revenue through a **commission-based model**.

---

## 📊 Dashboard Pages

The dashboard consists of multiple analytical pages:

### 1️⃣ Home Page

- Introduction to Uber platform  
- Navigation to different dashboard sections  
- Overview of the analysis structure  

---

### 2️⃣ Overview Dashboard

Key KPIs included:

- **Total Bookings:** 23K  
- **Lost Bookings:** 14K  
- **Revenue:** 13M  
- **Total Distance:** 625.62K km  
- **Average Ride Distance:** 24.62 km  

Additional insights:

- Monthly booking trends  
- Monthly revenue trends  
- Top drop locations  
- Customer and driver rating overview  

---

### 3️⃣ Vehicle Analysis

This section analyzes performance based on **vehicle types** such as:

- Auto  
- Bike  
- Go Mini  
- Go Sedan  
- Premier Sedan  
- Uber XL  

Metrics analyzed:

- Revenue generated by each vehicle  
- Number of completed bookings  
- Total ride distance  
- Lost bookings  

---

### 4️⃣ Cancellation Analysis

This page analyzes ride cancellations and incomplete rides.

Key insights include:

- **Total Bookings:** 2M  
- **Total Cancelled Rides:** 1K  
- **Incomplete Rides:** 321  
- **Total Ride Distance:** 87,110 km  

Major cancellation reasons:

- Change of plans  
- Wrong address  
- Driver asked to cancel  
- Driver not moving toward pickup  
- AC not working  

---

### 5️⃣ Rating Analysis

This section evaluates **customer satisfaction and driver performance**.

Key metrics:

- **Average Customer Rating:** 4.40 ⭐  
- **Average Driver Rating:** 4.23 ⭐  

Ratings are compared across vehicle types including:

- Auto  
- Bike  
- Go Mini  
- Go Sedan  
- Premier Sedan  
- Uber XL  

---

## 🛠 Tools & Technologies Used

- **Power BI** – Dashboard development and visualization  
- **Power Query** – Data cleaning and transformation  
- **DAX** – Calculated measures and KPIs  
- **Excel / CSV Dataset** – Data source  

---

## 📈 Key Insights

- Ride demand varies across months with noticeable fluctuations in bookings.  
- Some bookings are lost due to cancellations and incomplete rides.  
- Revenue contribution differs across vehicle types.  
- Customer ratings are generally higher than driver ratings.  
- Cancellation reasons highlight operational issues and user behavior patterns.
